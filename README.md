## Run Locally

Clone the project

```bash
  git clone https://github.com/patrykkawiak/RecruitmentApp.git
```

Go to the project directory

```bash
  cd {project_name}
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

The server will start at: http://localhost:3000/
